🚗 AI Car Advisor
Arabic NLP Intelligent Recommendation System






An AI-powered Arabic NLP application that provides intelligent and fuel-efficient car recommendations based on user input.

🌟 Project Overview

AI Car Advisor is a smart assistant that understands Arabic natural language queries and generates structured, context-aware car recommendations.

The goal of this project is to demonstrate how AI can move beyond theory into practical, real-world applications.

🧠 Key Features

✅ Arabic Natural Language Processing

✅ Intelligent recommendation logic

✅ Context-aware responses

✅ Structured output formatting

✅ Clean and interactive UI

⚙️ How It Works

User enters a query in Arabic

The system processes the natural language input

Intent and requirements are analyzed

Relevant fuel-efficient car options are selected

Structured recommendation is generated

🛠 Tech Stack

Python

Streamlit (UI Layer)

NLP Techniques

Data Processing (Pandas)

Structured Response Generation

▶️ Installation

Clone the repository:

git clone https://github.com/a7medabdullah7/ai-car-advisor-arabic-nlp.git
cd ai-car-advisor-arabic-nlp

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
